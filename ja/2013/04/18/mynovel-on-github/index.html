小説を２倍に拡大表示してくれる「吾輩の小説」のコードを公開

18 Apr 2013 / akira / 開発

昔のSF/推理小説がスマホで読めないと死んでしまう akiraak です。

そんな自分のために作ったiPhoneアプリ「吾輩の小説」がありました。既存のビュワーでは小説を画面に表示すると文字が小さくて読みづらいんですね。そこで作ったのが画像を２倍に拡大して表示するアプリです。行の真ん中で折り返すので、上下スクロールは必要なく、横スクロールの操作だけで読めます。

吾輩の小説

今回、こちらのコードをgithubに公開しました。 GitHub: akiraak/novel2x

技術的には画像のピクセルを解析して、行・文字を識別し、１行の半分で折り返して画面に表示するというものです。
